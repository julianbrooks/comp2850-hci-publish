Storage Instructions (For Researcher)

After obtaining consent:

  1. Scan or photocopy this form
  2. Give original or copy to participant for their records
  3. Store researcher copy in:
    • Physical: Locked filing cabinet in [location]
    • Digital scan: Encrypted university file storage (password-protected folder)
  4. Create key file linking Participant ID (P1, P2…) to name:
    • Store separately from research data
    • Password-protect
    • Access: Researcher only
  5. Retain for 6 years from study completion (university policy)
  6. After 6 years: Securely destroy (shred physical, permanently delete digital)

IMPORTANT: This consent form must be stored separately from research data to maintain pseudonymisation. Only the researcher should have access to the key file linking participant IDs to identifiable information.
